# Churches
A open-source database of US Churches compiled from IRS Exempt Organizations Business Master File. Enriched with matched website, biographical, theological, and service-related data

[Details about the data acquisition process and methodology can be found here](https://www.hotchkissmade.com/research/churches).

## 🙋🏽 I want to help!

For all of these, please open a `Discussion` above to reach out.

### Churches

First and foremost, the biggest way to help this project is just adding a page to your website with the ministries and small/home group options you have. Keep it up to date. Most churches have completely outsourced this role to Planning Center, who make it needlessly difficult to parse this information. 

There will eventually be an option to submit corrected information. The current limitation here is the ability to verify authority to present those details - this project does not aspire to be a OpenStreetMaps of churches as much as a representation of what churches advertise themselves as online.

### Theological

Once website matching has been completed, the next phase of the project is per-church LLM parsing to answer a list of about 75 questions for biography, theology, ministries offered, etc. Following that, transcribing sermons for LLM parsing to answer questions about church health.

Theological assistance (specifically from tech-forward seminary gradudates) is needed for some of the formation and reasoning aspects of this, especially for long term project questions about church health and flags.
